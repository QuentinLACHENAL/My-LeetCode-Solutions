# LeetCode Algorithms & Data Structures

Welcome to my personal LeetCode repository! 

This repository contains my solutions to various algorithmic problems and data structure challenges from [LeetCode](https://leetcode.com/). It serves as a log of my continuous learning, problem-solving practice, and interview preparation.

## Purpose
- **Consistent Practice:** Tracking my daily/weekly coding progress.
- **Problem Solving:** Improving my ability to break down complex problems and write efficient code.
- **Optimization:** Focusing on reducing Time ($O(n)$) and Space complexities.
- **Mastering Data Structures:** Deepening my understanding of Arrays, Hash Maps, Stacks, Trees, Graphs, and Dynamic Programming.

## Automated Sync
This repository is entirely automated! I set up a **GitHub Actions CI/CD workflow** that automatically fetches my latest accepted solutions directly from LeetCode and pushes them here. 
This allows me to focus 100% on solving problems on the platform while keeping my GitHub portfolio constantly up to date.

## Tech Stack
- **Primary Language:** Python 3
- **Automation:** GitHub Actions (`leetcode-sync`)

## Progress
The folders and files in this repository are generated automatically based on the problem title and my successful submissions. Each file contains the source code of the accepted solution.
